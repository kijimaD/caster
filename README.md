## 実行

```
make record
```
して、録画が終わったら`q`キーで抜ける。変換が実行されて、webpとwebmが生成される。

## ディレクトリ構造

一回ごとに保存・webp/webm化する。

```
- output
  - 20250101_23-30-55.webp
  - 20250101_23-30-55.webm
  - 20250102_09-20-19.webp
  - 20250102_09-20-19.webm
- photos
  - 20250101_23-30-55
    - 1.png
    - 2.png
    - 3.png
  - 20250102_09-20-19
    - 1.png
    - 2.png
```
